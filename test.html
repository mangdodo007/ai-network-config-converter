New Features Added:

  • ⚙️ Advanced Settings section (collapsible)
  • 🤖 AI Model Selection dropdown
  • 📝 Custom System Prompt textarea
  • 📊 Character counter for custom prompt
  • 🗑️ Clear custom prompt button

How to see the new UI:

  1. Open your browser and go to http://localhost:8002
  2. Look for the "Advanced Settings" section at the top (with ⚙️ icon)
  3. Click the "Show" button to expand the advanced settings
  4. You should see the Model Selection dropdown and Custom Prompt textarea

If you don't see the new UI, try:

  • Hard refresh (Ctrl+F5 or Cmd+Shift+R)
  • Clear browser cache
  • Open in an incognito/private window
